# Assignment 1
For this assignment, I have implemented all the "A" features, none of the A+ features are implemented. The controls are all as expected with `space bar` being used to switch between views and `q` for quit.  

The entire opengl-2022 sample project folder is included in this submission but all the assignment files are under `src/` directory, specifically `assignment1.cpp`, `vshaderA1` and `fshaderA1`. 

I did this assignment on macOS 26.1 (Tahoe) on a Macbook Air with the M1 chip. I used the makefile included in the `/src` folder to compile the program on my computer (macOS). This is the Makefile included in the sample project for mac, the `SRC_PREFIX` was just changed to be `assignment` here. 

## Build and Run
